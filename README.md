# Configuration

For the configuration, `config.js` and replace the :

```js
module.exports = {
  token: "",
  botID: "",
  ownerID: "",
  privateServerID: "",
  embedColor: "",
  ticketsSupportRoles: ["", ""],
  ticketsCategory: "",
  ticketsTranscripts: "",
};
```

# Commands

**Installation**

> `npm install`

**Installation**

> `node slash`

**Start the bot**

> `node index.js`
